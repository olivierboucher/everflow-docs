---
title: Generate Smart Link Tracking URL
openapi: /openapi/smart-links-extras.yaml post /networks/campaigns/{campaignId}/tracking
---

Generate a tracking URL for a specific smart link (campaign). Creates a trackable URL that routes traffic through the smart link's configured offer rotation and targeting rules.

Optionally provide `network_affiliate_id` and sub-parameters (`sub1`-`sub5`) in the request body.
