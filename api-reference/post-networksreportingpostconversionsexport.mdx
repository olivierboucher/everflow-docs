---
title: Export Post-Conversion Events
openapi: /openapi/reporting-exports.yaml post /networks/reporting/postconversions/export
---

Exports post-conversion event data in CSV or JSON format. This endpoint accepts the same parameters as the [Search Post-Conversion Events](/api-reference/post-networksreportingpostconversions) endpoint, with additional `format` and `columns` fields.

Post-conversion events are actions that occur after the initial conversion, such as upsells, renewals, or custom events.

Each request must contain:

- `from` and `to` dates (format can be `YYYY-MM-DD` or `YYYY-MM-DD HH:mm:SS`)
- The `timezone_id` used for the request
- The `currency_id` used for the reporting metrics
- The `format` field (`csv` or `json`)

The response is a file download with the appropriate `Content-Type` and `Content-Disposition` headers set.
