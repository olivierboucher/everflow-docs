---
title: List Blocked Sources
openapi: /openapi/affiliate-traffic.yaml get /affiliates/trafficblocking
---

View all blocked offer/sub-parameter combinations for your affiliate account. When a source is blocked, traffic from that combination will not generate valid clicks or conversions. Filter by offer ID or blocking status to find specific blocks.

Each entry includes the variables that define the blocking rule, such as the sub-parameter name, value, and comparison method used for matching. Use this endpoint to audit which traffic sources have been restricted for your account.
