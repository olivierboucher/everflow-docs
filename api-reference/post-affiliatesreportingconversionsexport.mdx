---
title: Export Conversions
openapi: /openapi/affiliate-reporting.yaml post /affiliates/reporting/conversions/export
---

Export conversion and event data in CSV or JSON format. Accepts the same parameters as the **Search Conversions** endpoint with an additional `format` field to specify the export type.

Use this endpoint when you need to download conversion data for external analysis, reporting tools, or record-keeping. The `format` field accepts `csv` or `json`.
