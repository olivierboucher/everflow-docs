---
title: Get Conversion
openapi: /openapi/affiliate-reporting.yaml get /affiliates/reporting/conversions/{conversionId}
---

Retrieve full details for a single conversion by its ID. Returns the complete conversion record including the associated click transaction ID, revenue, sale amount, conversion status, event name (for post-conversion events), sub-parameters, and related offer/geo/device data. Use this endpoint when you need to inspect a specific conversion from a postback callback or reporting output.
