title: Get Entity Report (Table)
openapi: /openapi/advertiser-reporting.yaml post /advertisers/reporting/entity/table
---

Retrieve a paginated list of aggregated reporting data in table format for the authenticated advertiser. Supports filters, sorting, and pagination to help you browse reporting data programmatically. Group by up to 10 dimensions with flexible filters. Maximum reporting interval is one year. Results limited to 10,000 rows.
