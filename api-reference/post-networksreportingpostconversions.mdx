---
title: Search Post-Conversion Events
openapi: /openapi/reporting-extras.yaml post /networks/reporting/postconversions
---

Returns post-conversion event data. Post-conversion events are actions that occur after the initial conversion, such as upsells, renewals, or custom events.

Each request requires `from`, `to`, and `timezone_id`. Set `show_events` to `true` to include event details in the response. Optional query filters can be added to narrow results.
