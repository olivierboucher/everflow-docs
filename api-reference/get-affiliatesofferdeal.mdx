---
title: Get Deal
openapi: /openapi/affiliate-deals.yaml get /affiliates/offers/{offerId}/deals/{dealId}
---

Retrieve a single deal by its ID within a specific offer. Use the optional query parameters to include additional relationship data such as locations, products, resources, and tracking configurations.

This endpoint is useful when you need the full detail view of a specific deal, including optional data that is not returned in the list endpoints. Returns 404 if the deal does not exist or is not visible to your affiliate account.
