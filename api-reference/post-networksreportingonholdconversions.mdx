---
title: Search On-Hold Conversions
openapi: /openapi/reporting-extras.yaml post /networks/reporting/onholdconversions
---

Returns conversions currently in on-hold status. On-hold conversions are pending review before being approved or rejected.

Each request requires `from`, `to`, and `timezone_id`. Optional query filters can narrow results by offer, affiliate, or other criteria. Use the PATCH endpoint to approve or reject on-hold conversions.
