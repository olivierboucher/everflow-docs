---
title: List Custom Landing Page Settings
openapi: /openapi/custom-landing-pages.yaml post /networks/custom/landingpage/table
---

Retrieve a paginated list of custom landing page settings. Supports search filters, sorting, and pagination to help you find and browse landing page configurations programmatically.

Custom landing pages allow networks to override the default offer URL for specific affiliates, directing their traffic to a different landing page.
