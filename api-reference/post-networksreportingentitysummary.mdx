---
title: Get Reporting Summary
openapi: /openapi/reporting-extras.yaml post /networks/reporting/entity/summary
---

Returns aggregated summary metrics without row-level detail. Provides a single summary object with totals for impressions, clicks, conversions, revenue, payout, profit, margin, and other key performance indicators.

Uses the same request format as the entity/table endpoint but returns only the overall summary instead of individual rows.
