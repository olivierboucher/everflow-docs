---
title: Get Impression URL
openapi: /openapi/affiliate-offers.yaml get /affiliates/offers/{offerId}/impressionurl/{urlId}
---

Generate an impression tracking URL for a specific offer. Impression URLs are used to track ad views separately from clicks, which is useful for display advertising and CPM-based campaigns. Use `urlId=0` when no extra destination URLs are needed.

This endpoint follows the same authorization rules as the tracking URL endpoint -- your affiliate account must be approved for the offer or the offer must be public.
