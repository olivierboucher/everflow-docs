---
title: Get Variance Report
openapi: /openapi/reporting-extras.yaml post /networks/reporting/variance
---

Compare reporting data between two time periods or between Everflow and advertiser data. Returns a table with variance percentages alongside current and comparison period metrics.

Each request requires both a current period (`from`/`to`) and a comparison period (`compare_from`/`compare_to`), along with `timezone_id`, `currency_id`, and at least one column.
