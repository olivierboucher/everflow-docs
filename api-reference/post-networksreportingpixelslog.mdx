---
title: Get Partner Pixel Log
openapi: /openapi/reporting-extras.yaml post /networks/reporting/pixels/log
---

Returns partner (affiliate) postback pixel firing log data. Shows when partner postback URLs were fired, their HTTP response status, and any errors. Useful for debugging postback integration issues with affiliates.

Each request requires `from`, `to`, and `timezone_id`. Optional query filters can narrow results.
