---
title: Create Reporting Adjustment
openapi: /openapi/reporting-extras.yaml post /networks/reporting/adjustments
---

Create a new reporting adjustment. Adjustments allow you to manually correct reporting metrics such as conversions, revenue, or payout for a specific date range, offer, and affiliate.

Provide the `network_offer_id`, `network_affiliate_id`, date range, and the adjustment values (conversions, revenue, payout) in the request body.
