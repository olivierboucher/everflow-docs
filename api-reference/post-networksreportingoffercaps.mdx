---
title: Get Offer Caps Report
openapi: /openapi/reporting-extras.yaml post /networks/reporting/offercaps
---

Returns current cap usage for one or more offers. Shows daily, weekly, monthly, and global cap consumption for conversions, clicks, revenue, and payout.

Provide an array of `offer_ids` and a `timezone_id` in the request body. The response includes cap usage details per offer.
