Advanced search endpoint for postbacks with support for search terms, filtering, and pagination. Returns results in a paginated table format, making it suitable for building postback management interfaces.

Use this instead of the basic list endpoint when you need pagination, need to search by keywords, or need to combine multiple filter criteria in a single request.
