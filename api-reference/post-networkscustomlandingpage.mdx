---
title: Create Custom Landing Page Setting
openapi: /openapi/custom-landing-pages.yaml post /networks/custom/landingpage
---

Create a new custom landing page setting. Custom landing pages allow you to override the default offer URL for specific affiliates, directing their traffic to a different landing page.

Provide the `network_offer_id`, `network_affiliate_id`, `url`, and `status` in the request body.
